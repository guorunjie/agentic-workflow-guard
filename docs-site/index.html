Scan workflows

Emit CI evidence

Quick start

npx agentic-workflow-guard scan . --format sarif --output awg.sarif
npx agentic-workflow-guard fix . --format json
npx agentic-workflow-guard rules verify .awg/rules/agentic-workflow-guard-core-rules.json
