GitHub Action Marketplace

Add AI workflow security scanning to pull requests.

Use the release-tagged action to generate SARIF, upload findings to GitHub Code Scanning, and publish a human-readable Step Summary.

Recommended workflow

name: agentic workflow guard

on:
  pull_request:
  push:
    branches: [main]

jobs:
  guard:
    runs-on: ubuntu-latest
    permissions:
      security-events: write
      contents: read
    steps:
      - uses: actions/checkout@v6
      - uses: guorunjie/agentic-workflow-guard@v0.15.0
        with:
          path: .
          format: sarif
          profile: balanced
          output: awg.sarif
        continue-on-error: true
      - uses: github/codeql-action/upload-sarif@v3
        with:
          sarif_file: awg.sarif

Inputs

`path`, `format`, `output`, `profile`, and `baseline` let teams move from advisory scans to strict enforcement.

Outputs

`report-path` points to the generated report so follow-up jobs can upload or archive it.

Artifacts

Use SARIF for Code Scanning, JSON for dashboards, and Markdown for local reviews or issue comments.

Use a pinned release tag for repeatable Marketplace installs.
